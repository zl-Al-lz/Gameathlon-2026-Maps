This repository contains my work on the official Age of Mythology: Retold competitive map scripts.

For Gameathlon 2026, I was responsible for implementing map balance improvements across the official tournament maps, including partial and complete script rewrites where needed. The event is an S-Tier LAN tournament held in Athens, Greece, featuring a $10,000 USD prize pool and officially sponsored by World's Edge (Microsoft).

The repository also includes the exclusive maps I developed for the previous Pandora's Box 2 release.
